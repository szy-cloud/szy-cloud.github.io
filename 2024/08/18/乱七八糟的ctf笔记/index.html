web

web前置知识

ip：找到你在哪里

端口：区分你是从哪里发出或接受的信息（qq或者wx）

html：标记型语言，可以将纯文本渲染成网页

协议：文件传输规范的格式。我们登录一个网页，如果用的是自己的账号密码，传输过去，返回回来的就是一个普通的页面，如果使用的是管理员的账号密码，那返回的就是管理员页面的格式。这种由传输的不同而引起返回的不同就是协议，即客户端计算机与服务端计算机的对话方式（就如data伪协议，用普通的方式传输payload，无法使服务端返回想要的东西。而用文件的格式传输，使payload伪装成文件，从而达到欺骗服务端的目的，返回的就是我们想要的）

cookie与session：传统的http协议为静态，服务端将每个请求都作为独立的请求，无法满足如今需要记住密码与账户的需求，于是我们将客户端与服务端之间的对话具体化成一个聊天窗口，这时服务端只需要调取上面的聊天记录便可记住账户密码。于是，cookie就是这个聊天窗口，而session便是聊天的内容

一、首先，尝试有无sql注入点

显而易见的是我们输入的东西是id的参数，加 ‘ 先闭合前面的引号，而这时后面会单独出现一个引号，如下所示

前置知识

XML：可扩展标记语言

应用于 Web 开发的许多方面，常用于简化数据的存储和共享。（HTML 旨在显示信息，而 XML 旨在传输信息。）

updatexml函数：updatexml(xml_doument,XPath_string,new_value)
第一个参数：XML的内容
第二个参数：是需要update的位置XPATH路径
第三个参数：是更新后的内容
所以第一和第三个参数可以随便写，只需要利用第二个参数，他会校验你输入的内容是否符合XPATH格式

当输入的XPATH不符合xml格式时会报错，此时页面会把报错的原因回显在页面上，方便管理者调试。于是就出现了报错注入

extractvalue函数：extractvalue(XML_document, XPath_string);
第一个参数：XML_document是String格式，为XML文档对象的名称；
第二个参数：XPath_string (Xpath格式的字符串)；
作用：从目标XML中返回包含所查询值的字符串；
用法与updatexml相同

语句：

sqlmap -r 文件名 –batch –dbs 爆数据库名

sqlmap -r 文件名 –batch -D 库名 –tables 爆表名

sqlmap -r 文件名 –batch -D 库名 -T 表名 –columns 爆字段名

sqlmap -r 文件名 –batch -D 库名 -T 表名 -C 字段名 –dump 爆指定数据库指定列指定字段下的全部数据

注意：

文件保存在桌面一定要在桌面打开终端

–batch是默认配置选项，不输这个就要选一大堆选项，千万不能去掉

-r是用于抓包保存于桌面的方式，-u是用于指定url地址的方式

反序列化

php反序列化

思路

利用各种魔术方法构建pop链

前置工作

找pop链时把变量全部改成public

常见魔术方法

__destruct：在到某个对象的所有引用都被删除或者当对象被显式销毁时执行。
（反序化前自动调用_destruct）

**__tostring()**：方法用于一个类被当成字符串时应怎样回应，出现情况：（这里的参数需被赋值为字符串）一个参数被用于正则匹配的时候，一个参数被echo，一个参数被函数调用的时候。

**__get()**：访问不存在的成员变量时调用。

**__wakeup()**：当在类外部使用unserialize()时会调用。

**__invoke()**：以调用函数的方式调用一个对象时会被调用。

**__set()**： 给一个未定义的属性赋值时，此方法会被触发。

**__call()**：当调用类实例中不存在的函数时自动执行。

注意！

①假如题目有私有变量，protect之类的，那么最后echo的时候需要对结果进行url编码，即
echo urlencode(serialize($a));

②题目有可能骗人，如果题目有提示使用伪协议，但执行到最后不在flag.php里，那就尝试/flag。

③做到最后一个函数一般有个拦路虎，目前情况有：伪协议、函数调用、rce。

④construct() 绝对不会出现在反序列化的题目里，所以__construct无需考虑进pop链中，所以！！！遇到construct直接删掉！！！！

⑤在西电的moectf里遇到的，如果遇见一些题，他有一大堆私有和保护变量导致代码写一半经常各种报错（类似于创建错误的类，类必须是string啊什么的），那就只能用以下办法构造pop链了

反序列化相关知识点

php引用赋值 &

这个的作用是用来达成一些特定的判断，拿一道题举例

这是一道反序列化的题目，做题的时候首先是知道了pop链，要从Hacker里的destruct去到C里面的tostring

让我们先来看看tostring的定义

**__tostring()**：方法用于一个类被当成字符串时应怎样回应

之前对tosrting的理解还是过于浅薄，因为以前做的题目都是用一个参数将他实例化成对象然后在别的魔术方法里进行正则然后就触发了tostring，但是在这道题里，username参数他是在wakeup里赋值成了一个字符串，但是wkeup在这个版本的php里是没办法用改对象属性个数的方式绕过的，所以只能直面wakeup，如果用username去实例化然后往下走，在wakeup里还是会重新变成字符串

所以我们只能在username被赋值为字符串之后再重新将它变成实例化的对象，注意到赋值的代码往下走后，有一个将start的值赋给end的操作，所以这时候就需要用到引用赋值

如果我们将end和username联系起来，那么修改start了是不是也同样修改了username呢？

那么接下来就来看看引用赋值怎么用了

可以看到，用法是$b = &$a

那么在这道题里，将end和username联系起来就是$end=&$username

从某种意义上也是变相绕过wakeup了

wakeup绕过：

1、PHP5 < 5.6.25 ，PHP7 < 7.0.10：序列化字符串中表示对象属性个数的值大于真实的属性个数时会跳过wakeup的执行，所以在本地调试题目出来的反序列化代码那一长串上更改属性个数即可跳过wakeup，类似于曾经遇见的wakeup里面带有hax加密，目前没有解密hax的办法，于是需跳过__wakeup ）。

2、gc回收

gc：garbage collection，在php中，一个参数在没有任何引用的情况下会被当成垃圾回收掉；同理，一个对象在没有任何引用的情况下也会被当成垃圾

RCE（ping命令）

rce分为代码执行和命令执行

代码执行：给一长串代码，然后某个地方出现eval函数，这种情况就需要用到system和passthru了

命令执行：一般是给一个很正经的页面然后有一个框给你输入，这时候就只需要直接输入Linus命令就可以了。但需注意的是需要在执行的命令前面加上分号（或空格或回车（?>））截断前面他规定你输入的东西。比较典型的例子就是ping命令，需要在前面输入ip然后用分号截断，之后再接上Linus命令。
